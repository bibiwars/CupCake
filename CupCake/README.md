    ## API Result codes
        0: no errors
        1: unknown error
        1000: unauthorized
        1001: password value empty
        1002: password verification does not match
        1003: old password does not match
        1004: login failed
        1005: user not found
        1006: user value empty
        1007: cibletype value invalid
        1008: cibleid value invalid
        1009: note value invalid
